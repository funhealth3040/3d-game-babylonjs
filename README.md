# react-meteor-redux-eui-boilerplate
React, Meteor, Redux Observable, Elastic UI, SCSS, Final Form Boilerplate

# install
```
// meteor 설치 
$ curl https://install.meteor.com | sh
// yarn 설치
$ meteor npm i -g yarn
// 패키지 설치 
$ meteor yarn 
```

# run
```
$ meteor run
// browser
http://localhost:3000
```
